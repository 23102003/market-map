# JSW Market Map

Internal district-wise brand market mapping tool for JSW Steel.

## What This Does
- Shows district-level brand volume across states
- Auto-updates daily from SAP dispatch data
- Built with Python + Streamlit

## Daily Update Process
1. Get SAP dispatch Excel file
2. Upload to this repository
3. Map auto-refreshes within 2 minutes

## Files
| File | Purpose |
|------|---------|
| state_colour_frame.py | Main Streamlit app |
| auto_populate_market_map.py | Data processing script |
| market_map_data.py | Auto-generated data (do not edit) |
